# ウォレット接続

暗号通貨ウォレットはMUSEのブロックチェーン機能に必要ですが、それらの機能のみに限られます。MUSEが提供する機能の大部分は、ウォレットなしで問題なく動作します。このガイドでは、ウォレットが必要な場面、接続方法、初めての方向けのセットアップ手順を説明します。

## ウォレットは必要ですか？

**ほとんどの機能ではウォレット不要：**

| 機能 | ウォレットが必要？ |
|------|-----------------|
| エビデンスの閲覧と検索 | 不要 |
| エビデンス詳細ページの閲覧 | 不要 |
| キャンバスでのロジックモデル構築 | 不要 |
| ノードと接続の編集 | 不要 |
| AIによるロジックモデル生成 | 不要 |
| IPFSへの保存と共有 | 不要 |
| 画像またはJSONとしてエクスポート | 不要 |
| 共有キャンバスの閲覧 | 不要 |
| ハイパーサートギャラリーの閲覧 | 不要 |

**これらの機能にはウォレットが必要：**

| 機能 | ウォレットが必要？ |
|------|-----------------|
| ハイパーサートのミント | 必要 |
| オンチェーンアテステーションの署名 | 必要 |

:::tip ウォレットなしで始められます
MUSEが初めての方は、すぐにウォレットをセットアップする必要はありません。キャンバスを探索し、AIでロジックモデルを生成し、エビデンスデータベースを閲覧しましょう。これらすべてはウォレットなしで利用できます。ハイパーサートをミントする準備ができたら、ウォレットのセットアップに戻ってきてください。
:::

## 暗号通貨ウォレットとは？

暗号通貨ウォレットは、一対の暗号鍵を管理するソフトウェアです。以下のことができます：

- ブロックチェーントランザクション（ハイパーサートのミントなど）に署名して、あなたが承認したことを証明する
- 暗号通貨を保有する（「ガス代」と呼ばれるトランザクション手数料の支払いに使用）
- MUSEのような分散型アプリケーションとやり取りする

ウォレットを使うために技術的な詳細を深く理解する必要はありません。ブロックチェーンアプリケーションで本人確認に使うデジタルIDカードのようなものだとお考えください。

## 対応ウォレット

MUSEは **RainbowKit** を使用しており、幅広い人気ウォレットに対応しています：

- **MetaMask** — 最も広く使われているブラウザ拡張ウォレット（[metamask.io](https://metamask.io)）
- **Coinbase Wallet** — Coinbaseが提供するブラウザ拡張およびモバイルアプリ
- **WalletConnect** — QRコードをスキャンして多くのモバイルウォレット（Trust Wallet、Rainbowなど）に接続できるプロトコル
- **インジェクトウォレット** — ブラウザ拡張としてすでにインストールされているウォレットは自動的に選択肢として表示されます

まだウォレットをお持ちでない場合は、MetaMaskから始めるのがおすすめです。[metamask.io](https://metamask.io) にアクセスしてブラウザ拡張をインストールし、セットアップ手順に従って新しいウォレットを作成してください。

:::warning シードフレーズを安全に保管してください
新しいウォレットを作成すると、**シードフレーズ**（リカバリーフレーズとも呼ばれます）— 12語または24語の単語リスト — が発行されます。これはウォレットのマスターキーです。紙に書き留めて安全な場所に保管してください。誰にも教えないでください。ウェブサイトに入力しないでください。シードフレーズを知っている人は、あなたのウォレットに完全にアクセスできてしまいます。
:::

## ウォレットの接続方法

1. MUSEナビゲーションバーの右上にある **「Connect Wallet」** ボタンをクリックします
2. 利用可能なウォレットオプションを表示するダイアログが表示されます
3. ウォレットプロバイダーをクリックします（例：MetaMask、WalletConnect）
4. ウォレットソフトウェアが開きます（WalletConnectの場合はQRコードが表示されます）
5. ウォレット内で接続リクエストを確認し、**「Connect」** または **「Approve」** をクリックします
6. 承認が完了するとダイアログが閉じ、接続されたウォレットアドレスがナビゲーションバーの右上に表示されます

これで接続完了です。MUSEは現在のセッション中、ウォレットの接続を記憶します。

## 接続状態の確認

接続後、ナビゲーションバーには以下が表示されます：

- **ウォレットアドレス** — 短縮形式で表示されます（例：`0x1234...5678`）
- **残高** — 現在のネットワーク上のウォレット内のETH残高

ウォレットの残高は、ハイパーサートのミントやアテステーションの署名時にガス代を支払うために使用されます。

## ネットワーク設定

MUSEは使用しているアプリのバージョンに応じて、自動的に適切なネットワークに接続します：

| アプリ | URL | ネットワーク | 意味 |
|--------|-----|------------|------|
| 本番環境 | [muse.beaconlabs.io](https://muse.beaconlabs.io) | Ethereum（本番ネットワーク） または Base | 実際のトランザクション、実際のガス代 |
| 開発環境 | [dev.muse.beaconlabs.io](https://dev.muse.beaconlabs.io) | Base Sepolia（テスト用ネットワーク） | テストトランザクション、実際のコストなし |

ウォレットが間違ったネットワークに接続されている場合、MUSEはネットワークの切り替えを求めるプロンプトを表示します。プロンプトをクリックし、ウォレット内でネットワーク切り替えを承認してください。

## 開発環境用テスト用ETHの取得

開発環境（[dev.muse.beaconlabs.io](https://dev.muse.beaconlabs.io)）を使用している場合、テストトランザクションの支払いのために **Base Sepolia** ネットワーク上のテストETHが必要です。テストETHには実際の価値はありません — 練習用です。

無料のテストETHを取得するには：

1. 開発環境アプリにウォレットを接続します
2. ウォレットが **Base Sepolia** ネットワークになっていることを確認します（なっていない場合はMUSEがプロンプトを表示します）
3. Sepoliaフォーセットにアクセスします — ブラウザで「Base Sepolia faucet」と検索して現在の選択肢を見つけてください
4. ウォレットアドレスを入力してテストETHをリクエストします
5. 1〜2分待ってテストETHがウォレットに届くのを確認します
6. テストハイパーサートをミントする準備が整いました

:::tip フォーセットは時間とともに変わります
フォーセットのURLは定期的に変更されます。見つけたフォーセットが機能しない場合は、新しいリストを検索してください。[Baseネットワークのドキュメント](https://docs.base.org)で現在のフォーセットの推奨情報も確認できます。
:::

## ウォレットの切断

MUSEからウォレットを切断するには：

1. ナビゲーションバー右上のウォレットアドレスをクリックします
2. ドロップダウンまたはモーダルが表示されます
3. **「Disconnect」** を選択します
4. ウォレットがMUSEから切断されます

切断すると、現在のブラウザセッションにおけるMUSEとウォレットの接続が解除されます。ウォレット自体には影響ありません — 資金と鍵は安全なままです。

## よくある質問

**ウォレットのデータはMUSEに保存されますか？**
いいえ。MUSEがあなたの秘密鍵やシードフレーズにアクセスすることはありません。ウォレットを「接続」する際、MUSEが取得するのはあなたの公開ウォレットアドレスのみです。これはブロックチェーン上で誰でも確認できる情報と同じです。すべての署名はウォレットソフトウェア自体で行われます。

**ハードウェアウォレット（Ledger、Trezor）は使えますか？**
はい、WalletConnect経由で使用できます。WalletConnectオプションを選択し、ハードウェアウォレットのペアリング手順に従ってください。

**接続ダイアログにウォレットが表示されません。**
ウォレットのブラウザ拡張がインストールされ、ロック解除されていることを確認してください。モバイルウォレットを使用している場合は、WalletConnectオプションを選択してスマートフォンでQRコードをスキャンしてください。

**トランザクションが失敗しました。どうすればいいですか？**
トランザクションの失敗は通常、ガス不足（手数料に必要なETHが足りない）またはネットワークの混雑が原因です。ETHの残高を確認して再試行してください。開発環境の場合は、フォーセットから追加のテストETHを取得してください。
