# はじめに

MUSEへようこそ — [Beacon Labs](https://beaconlabs.io) が開発した、エビデンスに基づくインパクト計画ツールです。MUSEを使うと、査読済みの研究エビデンスとブロックチェーンによる検証済みアテステーションを組み合わせたロジックモデル（セオリー・オブ・チェンジ）を作成できます。

## MUSEとは？

MUSE（Modular Stack of Evidence）は、研究エビデンスと現実世界のインパクト計画との間にある隔たりを埋めます。MUSEでは以下のことができます：

- **ビジュアルなロジックモデルの作成** — プログラムのアクティビティが測定可能なインパクトにどうつながるかをマッピングする
- **AIの活用** — 取り組みの説明を入力するだけで、ロジックモデルを自動生成する
- **研究エビデンスのリンク** — 査読済みの研究を、ロジックモデルの各ステップに関連付ける
- **ブロックチェーンによる検証** — Hypercert（ハイパーサート）とEASアテステーションを通じて、インパクトの主張を変更不可能・検証可能な形で記録する

## MUSEは誰のためのもの？

MUSEは、インパクトの計画と評価に携わるすべての人に向けて設計されています：

- **NGO・非営利団体** — エビデンスに基づくプログラムの立案
- **研究者** — 研究成果を実践的な応用につなげる
- **インパクト投資家** — 研究的裏付けを持つプログラム設計の評価
- **政策立案者** — エビデンスに基づく意思決定
- **デジタル公共財コミュニティ** — インパクトの追跡と検証

## MUSEにアクセスする

### 本番環境

実際のインパクト計画やブロックチェーンアテステーションには、メインのアプリケーションを使用してください：

- **URL**: [https://muse.beaconlabs.io](https://muse.beaconlabs.io)
- **用途**: 本番用ロジックモデル、実際のHypercert、永続的なエビデンスアテステーション

### 開発環境

実際のブロックチェーンのコストをかけずに練習・実験ができます：

- **URL**: [https://dev.muse.beaconlabs.io](https://dev.muse.beaconlabs.io)
- **ネットワーク**: Base Sepolia（テスト用ネットワーク）
- **用途**: 学習、機能のテスト、Hypercertのミント練習

:::tip 初めて使う方へのおすすめ
まずは**開発環境**でMUSEの機能を費用なしに試してみましょう。使い方に慣れたら、実際のインパクト活動に本番アプリへ移行してください。
:::

## クイックスタート：最初の5分間

1. **アプリにアクセス** — [muse.beaconlabs.io](https://muse.beaconlabs.io)（または[開発環境](https://dev.muse.beaconlabs.io)）を開く
2. **キャンバスに移動** — 上部ナビゲーションバーの「Canvas」をクリック
3. **AIで生成** — 「Generate from Intent」ボタンをクリックし、取り組みの内容を入力する
   - 例：*「OSSのEthereumエコシステムへのインパクト」*
   - 例：*「都市コミュニティの食料不安の軽減」*
4. **ロジックモデルを確認** — 約40秒後に以下が表示されます：
   - 各ステージを表す色分けされたカード（アクティビティ → アウトプット → アウトカム → インパクト）
   - ステージ間の因果関係を示す矢印
   - エビデンスに裏付けられた接続を示す緑色のエッジ
5. **エビデンスを確認** — 緑色のエッジをクリックすると、各接続を支持する査読済み研究が表示される

以上です！エビデンスに基づくセオリー・オブ・チェンジが完成しました。
