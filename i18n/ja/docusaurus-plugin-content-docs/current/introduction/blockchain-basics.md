# ブロックチェーンの基礎

MUSEはブロックチェーン技術を使用して、インパクトの主張を透明かつ誰でも独立して検証できる形にします。このページでは、暗号通貨やブロックチェーンの事前知識がなくても理解できるよう、主要な概念をわかりやすく解説します。

:::info MUSEのほとんどの機能にウォレットは不要です
MUSEのコア機能 — エビデンスの閲覧、キャンバスでのロジックモデル構築、AIによるモデル生成、IPFSへの保存 — はすべて暗号通貨ウォレット**なし**で利用できます。ウォレットが必要になるのは、ハイパーサート（オンチェーンのインパクト証明書）をミントしたい場合のみです。ウォレットをセットアップする前に、MUSEの機能をすべて自由に試してみてください。
:::

## なぜMUSEはブロックチェーンを使うのか？

インパクトの主張は、その背後にあるプロセスと同程度の信頼性しか持ちません。従来のドキュメントは密かに編集されたり、削除されたり、日付を遡って作成されたりする可能性があります。ブロックチェーンはこの問題を解決します。記録が以下のような台帳に保存されるからです：

- **透明性** — 誰でも記録を閲覧できる
- **不変性** — 一度書き込まれた記録は変更や削除ができない
- **検証可能性** — 誰でも記録が存在し、特定の時刻に作成されたことを独立して確認できる

MUSEでは、ブロックチェーンを使用してエビデンスがレビューされアテステーションされたことを検証し、ハイパーサートを通じてインパクト主張の永続的・譲渡可能な記録を作成します。

## 暗号通貨ウォレットとは？

暗号通貨ウォレットは、ブロックチェーンネットワークと対話できるデジタルツールです。安全なデジタルIDのようなものと考えてください — ウォレットは、トランザクションに署名しデジタル資産の所有権を証明できる固有のアドレス（メールアドレスに似たもの）を提供します。

お金をウォレットの「中に」保存するわけではありません。ウォレットは、ブロックチェーン上のアドレスを管理していることを証明する秘密鍵を保持しています。一般的なウォレットのオプションには以下があります：

- **MetaMask** — ブラウザ拡張機能とモバイルアプリ；最も広く使われているウォレット
- **Coinbase Wallet** — 初心者にやさしいインターフェースのもう一つの人気ウォレット
- **Rainbow** — クリーンなデザインのモバイルファーストウォレット

MUSEにウォレットを接続する際は、アプリにあなたのIDを伝えて、インパクトの主張があなたのアドレスに紐付けて記録されるようにしているだけです。MUSEが秘密鍵やシードフレーズを求めることは決してありません — それはウォレットアプリのみが行うことです。

:::caution シードフレーズを保護してください
ウォレットのシードフレーズ（12語または24語のフレーズ）は、ウォレットのマスターキーです。絶対に誰とも共有したり、どのウェブサイトにも入力したりしないでください。MUSEがシードフレーズを求めることは決してありません。
:::

## 本番環境 vs. 開発環境：どちらを使うべきか？

MUSEは2つの独立したネットワークで動作しています：

| | 本番環境 | 開発環境 |
|-|---------|------------|
| **URL** | [muse.beaconlabs.io](https://muse.beaconlabs.io) | [dev.muse.beaconlabs.io](https://dev.muse.beaconlabs.io) |
| **ネットワーク** | Base（本番ネットワーク） | Base Sepolia（テスト用ネットワーク） |
| **ガス代** | 少額の実費 | 無料 |
| **ハイパーサート** | 永続的・実際の記録 | テスト記録のみ |
| **最適な用途** | 実際のインパクト活動 | 学習と実験 |

**Base Sepolia**はBaseブロックチェーンのテスト版で、トランザクションは無料かつ永続的ではありません。実際のトランザクションを行う前に、ウォレットとハイパーサートの使い方を学ぶのに最適な場所です。

ブロックチェーンが初めての方は、**開発環境**から始めることをおすすめします。Base Sepolia faucetからテスト用資金（「テスト用ETH」）を無料で入手し、コスト無しのテストトランザクションに使用できます。

## EAS：Ethereum Attestation Service

**アテステーション**とは、ある当事者が別の当事者について署名した声明です — 推薦状に似ていますが、ブロックチェーン上に書かれるため偽造や変更ができません。

MUSEは**EAS（Ethereum Attestation Service）**を使用して、ライブラリ内のエビデンスの有効性をアテステーションします。エビデンスカードが検証・承認されると、Base SepoliaブロックチェーンにEASアテステーションが作成されます。このアテステーションには以下が記録されます：

- **何を** — エビデンスのコンテンツハッシュ（フィンガープリント）
- **いつ** — アテステーションが作成された正確な日時
- **誰が** — アテステーションを作成した当事者のアドレス

誰でもパブリックなブロックチェーンエクスプローラーでこのアテステーションを検索し、特定の時点に特定の形式でエビデンスが存在していたことを確認できます。これは従来のドキュメントでは不可能なレベルの説明責任を提供します。

## IPFS：分散型ファイルストレージ

**IPFS**（InterPlanetary File System）は、ファイルを分散型で保存・共有するためのネットワークです。従来のサーバー（ファイルが1か所に存在し、サーバーがオフラインになると消える）とは異なり、IPFSは世界中の多くのコンピュータにファイルを分散させます。

MUSEでロジックモデルを保存したり、エビデンスが検証されたりすると：

1. コンテンツがIPFSにアップロードされる
2. IPFSがユニークな**コンテンツ識別子（CID）** — ファイルの内容から数学的に導出されたハッシュ — を生成する
3. このCIDが永続的で共有可能なリンクとして使用される

CIDはコンテンツ自体から導出されるため、ファイルに少しでも変更を加えると完全に異なるCIDが生成されます。これにより、検出されることなくコンテンツを変更することが不可能になります。

MUSEでは、IPFSリンクを使ってロジックモデルを誰とでも共有できます — MUSEアカウントを持っていない人でも、永続的なURLを通じてアクセスできます。

## ハイパーサート：ブロックチェーン上のインパクト証明書

**ハイパーサート**は、インパクトの主張を表すデジタル証明書です。良い仕事をした証拠 — 「このプログラムが、この期間に、このアウトカムに貢献した」という永続的・公開的な記録 — と考えることができます。

MUSEのハイパーサートには以下が記録されます：

- **活動の範囲** — どのようなアクティビティが実施されたか
- **期間** — いつ活動が行われたか
- **貢献者** — 誰が活動を行ったか
- **ロジックモデル** — アクティビティからインパクトへの、エビデンスに裏付けられた経路

一度ミントされたハイパーサートはブロックチェーン上に永続的に存在します。公開共有、助成金申請での参照、将来のインパクト資金調達メカニズムでの活用が可能です。ハイパーサートは相互運用性を考慮して設計されており — 他のプラットフォームが認識し活用できるオープンスタンダードに従っています。

ハイパーサートをミントするには、ウォレットの接続が必要です。詳細な手順については[ハイパーサートガイド →](../user-guide/hypercerts)をご覧ください。

## ガス代：ブロックチェーントランザクションのコスト

ブロックチェーン上のすべてのトランザクション（アテステーションの作成やハイパーサートのミントなど）には、**ガス代**と呼ばれるネットワークへの少額手数料が必要です。ガス代は、トランザクションを処理・検証するコンピュータへの報酬です。

MUSEでは：

- **開発環境のトランザクション** — 無料。開発環境は価値のないテストトークンを使用するため、実際のコストは発生しません。
- **本番環境のトランザクション** — ETH（Ethereum）での少額手数料が必要です。Baseネットワークでは、これらの手数料は通常非常に少額（多くの場合、数セント程度）です。

エビデンスを閲覧したり、ロジックモデルを構築したり、IPFSに保存したりするだけではガス代は発生しません。ガスが課金されるのは、ブロックチェーンにトランザクションを書き込む場合のみです — 例えば、ハイパーサートをミントする場合です。

## まとめ：ウォレットが必要な機能

| 機能 | ウォレットの必要性 |
|------|-----------------|
| エビデンスライブラリの閲覧 | 不要 |
| キャンバスでのロジックモデル構築 | 不要 |
| AIによるロジックモデル生成 | 不要 |
| エビデンスの検索と探索 | 不要 |
| ロジックモデルのIPFS保存 | 不要 |
| リンクによるロジックモデルの共有 | 不要 |
| ハイパーサートのミント | **必要** |
| EASアテステーションの作成 | **必要** |

今すぐMUSEを試してみましょう — ウォレットは不要です。インパクトの主張をオンチェーンにする準備ができたら、[ウォレット接続ガイド →](../user-guide/wallet-connection)でセットアップ方法を順を追って説明しています。
